# 项目介绍
基于Spring AI Alibaba Graph 实现的小红书笔记工作流，包括数据采集、数据处理、数据存储、数据发布等功能。
- 数据采集：从指定的数据源采集数据，保存为md文件。默认数据源 HackNews (https://news.ycombinator.com/)
- 数据处理：利用模型对采集到的数据进行处理，包括标题生成、摘要提取、标签分类。通过python脚本和html模版将处理后的数据转换为png图片
- 数据存储：将处理后的数据存储到数据库中
- 数据发布：根据处理结果，发布到小红书或者其他平台（后续补充）
- 通知：采集、处理、发布数据后，通过邮件通知用户（后续补充）

项目根目录下的out目录：保存采集到的数据、处理后的图片以及日志信息

# 环境
- JDK 17+
- Python 3.11+
- 模型：https://bailian.console.aliyun.com 百炼平台或者Ollama本地模型
